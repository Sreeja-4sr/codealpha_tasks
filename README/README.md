### Objective
Build a machine learning model to classify customer reviews as Positive, Neutral, or Negative.

### Tools Used
- Python
- Pandas

- TF-IDF Vectorizer
- Logistic Regression
- Matplotlib
- Seaborn

### Workflow

- Data Cleaning
- Sentiment Label Creation

- Train-Test Split
- Logistic Regression Model
- Model Evaluation
- Confusion Matrix

### Evaluation Metrics
- Accuracy Score
- Classification Report
- Confusion Matrix

### Skills Learned
- Natural Language Processing (NLP)
- Text Preprocessing
- Feature Extraction
- Machine Learning
- Model Evaluation
